# Prisma ORM
This template uses [Prisma](https://www.prisma.io/) for its ORM. 

Reasons for selecting Prisma:
- Type Safety
- Easy to understood

## Useful Commands
- Seed the DB
```bash
yarn db:seed
```
- Run migrations: 
```bash
yarn db:migrate:dev
```
- Generate a new PrismaClient: 
```bash
yarn db:generate
```
